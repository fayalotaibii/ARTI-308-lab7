# Advertising Click Prediction - Logistic Regression

This project utilizes **Logistic Regression** to analyze customer data and predict whether an internet user will click on an advertisement based on their demographics and browsing behavior.

## Project Overview
The dataset contains behavior metrics such as session length, age, area income, and daily internet usage. We aim to predict the **Clicked on Ad** feature to help a company optimize its advertising strategy.

## Tech Stack
* **Language:** Python
* **Libraries:** `pandas`, `numpy`, `seaborn`, `matplotlib`, `scikit-learn`

## Key Findings
Based on the Logistic Regression model, the following patterns were observed:
* **High Engagement:** Users with lower daily internet usage and lower time spent on the site were statistically more likely to click the advertisement.
* **Demographics:** Age and Area Income also played significant roles in the classification.
